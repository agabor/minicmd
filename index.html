YACT

Yet Another Coding Tool - A command-line AI assistant powered by Claude

Get Started

Installation

Download the latest binary for your operating system:

Linux
yact-linux-amd64
Windows
yact-windows-amd64.exe
macOS
yact-darwin-amd64 / arm64

Get binaries from GitHub releases

Linux Example

cd /tmp
wget https://github.com/agabor/yact/releases/download/v0.0.0/yact-linux-amd64
chmod +x yact-linux-amd64
sudo mv yact-linux-amd64 /usr/local/bin/y

macOS Example

cd /tmp
curl -L -o yact-darwin-amd64 https://github.com/agabor/yact/releases/download/v0.0.0/yact-darwin-amd64
chmod +x yact-darwin-amd64
sudo mv yact-darwin-amd64 /usr/local/bin/y

Quick Start

Configure API Key

y config anthropic_api_key YOUR_API_KEY

Verify Configuration

y config

Commands Overview

Generate Code

Create code files from prompts with full file generation.

Bash Scripts

Generate standalone bash scripts from descriptions.

Ask Questions

Get answers about your codebase or general topics.

File References

Attach files with glob patterns for AI context.

Core Commands

Generate Code

y act "create a user authentication handler"

With safe mode (adds .new suffix for review):

y act --safe "add logging to the user service"

Generate Bash Scripts

y bash "create a script that backs up my database"

Ask Questions

y ask "how does the authentication flow work?"
y ask "what are the best practices for Go error handling?"

File References

y read src/models/user.go
y read src/handlers/*.go
y ask "add validation to the user model"

Context Management

y list

View current attachments

y clear

Clear all attachments

y new

Start a fresh conversation

y last

Retrieve the last AI response

Configuration

Available configuration keys:

Key Description Default
anthropic_api_key Your Claude API key (required) -
claude_model Which Claude model to use claude-haiku-4-5-20251001

Set Configuration

y config anthropic_api_key your_key_here
y config claude_model claude-opus-4-1-20250805

Usage Tips

  • File patterns: Use glob patterns with read to attach multiple files
  • Safe mode: Always use --safe when generating code to review first
  • Context: Read relevant files before asking complex questions
  • Conversation flow: Ask for a plan first, then generate code
  • Cost tracking: Tool displays token usage and estimated costs

Piping Input

cat requirements.txt | y act
echo "fix the database connection" | y bash

Storage & Troubleshooting

Storage Location

Configuration and conversation history are stored in ~/.yact/:

  • config - API key and model settings
  • context.json - Conversation history
  • attachments.json - List of attached files

Common Issues

  • "Claude API key not configured" - Run: y config anthropic_api_key YOUR_KEY
  • "No files found matching pattern" - Verify glob pattern matches existing files
  • "File not found in attachments" - Use y list to see attached files
  • API errors - Verify API key validity and internet connection
